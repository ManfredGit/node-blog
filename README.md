# node-blog-api

This is just a simple API written in Express and TypeScript for a blog.

### Get Started:
``` git clone https://github.com/iamclaytonray/node-blog-api.git ```

``` cd node-blog-api ```

``` yarn install ```

``` tsc ```

``` npm run start ```

That's it!
